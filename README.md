# google-cloud-conference-website



GitHub README
Google Cloud Conference Website using Google Antigravity (Generative AI)
Project Overview
This project was developed during the SmartBridge Google Cloud Generative AI Internship. The objective was to build an interactive conference website using Google Antigravity, an AI-assisted development environment powered by Gemini, along with Flask, HTML, CSS, and JavaScript.
The application enables users to browse conference sessions, search by speaker or topic, and filter sessions using category buttons. AI-assisted development significantly reduced development time while improving productivity.
